# Podium Pal

**Your Personal AI Public Speaking Coach**

## Overview

Podium Pal helps you become a better public speaker by providing real-time feedback on your speech. Practice your presentations and get instant insights on pacing, filler words, and our unique Clarity Score.

## Tech Stack

- **Frontend:** React with Vite
- **Backend:** Python, FastAPI
- **AI:** Google Gemini API

## Quick Start

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Then visit `http://localhost:5173`

## Usage

1. Enter your speech goal in the input field
2. Click "Start Recording" and begin speaking
3. Click "Stop Recording" when finished
4. Receive instant AI-powered feedback!
